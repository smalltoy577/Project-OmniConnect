Network Topology Design & Implementation
This project demonstrates a hierarchical enterprise network designed using Cisco Packet Tracer. It simulates a multi-departmental infrastructure with a focus on routing, switching, and server management.

Key Features & Architecture:
Hierarchical Routing: Utilizes a gateway router (R-Gateway) connected to a cloud service, managing traffic between the Academic and Admin departments.

Redundancy & Switching: Includes dedicated switches for different sectors (SW-Admin, SW-Academic, and SW-Gateway) to ensure efficient local traffic handling.

Server Infrastructure: Features centralized services including a DNS Server and a Database Server for network-wide resource access.

End-Device Connectivity: Supports a variety of endpoints including desktop PCs, laptops, a network printer, and a wireless segment.

Wireless Access: An Access Point (Uni WiFi) provides connectivity for smartphones and mobile laptops within the Academic sector.

Technical Skills Demonstrated:
VLAN segmentation (Admin vs. Academic).

Subnetting and IP address planning (e.g., 10.44.0.0/24 network).

Configuration of Wireless LAN (WLAN).

Server roles configuration (DNS and Database).
